----------------------------------------------------------------------------------------
------------------------------ Anagram Read-me -----------------------------------------
----------------------------------------------------------------------------------------

------
Run
------
1)for main program type into command line

	$node anagram.js

2)for test type into command line

	$node anagram_test.js

------
Output
------
Anagram.js is intended to take in  alist of words and return a list of sets of anagrams
present in input, ordered by size of set.

--------
Comments
--------
1) anagram.js works with node.js from command line
2)program assumes existence of /usr/share/dict/words
	~if target file is different change accordingly within anagram.js	
3)anagram_test.js runs and checks existence of output file out.txt
